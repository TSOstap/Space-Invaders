# Space Invaders (HTML5 Canvas)

## Опис гри
`Space Invaders` — це аркадна 2D-гру у ретро-стилі, зроблена на `HTML5 Canvas` і `Vanilla JavaScript`.  
Гравець керує космічним кораблем, рухається вліво/вправо, стріляє по формації прибульців, ухиляється від ворожих пострілів і набирає очки.  
У грі є:
- система очок і життів;
- прогресивне прискорення ворогів;
- ефекти вибухів (particles);
- звук через `Web Audio API`;
- збереження рекорду через `localStorage`.

## Інструкції як запустити
1. Завантажте або відкрийте папку проєкту.
2. Запустіть у браузері один із файлів:
   - `space-invaders.html` (базова версія);
   - `space-invaders-nova.html` (розширена версія).
3. Керування:
   - `A/D` або `←/→` — рух;
   - `Space` — постріл;
   - `Enter` — старт/рестарт;
   - `P` — пауза.

## Опис процесу створення
Процес розробки складався з таких етапів:
1. Створення базової архітектури гри в одному HTML-файлі (`canvas`, HUD, game loop).
2. Реалізація основної логіки:
   - гравець, вороги, кулі, колізії;
   - стани гри (`ready`, `playing`, `paused`, `gameover`);
   - система очок, життів і high score.
3. Додавання візуалу та атмосфери:
   - зоряний фон, неоновий стиль;
   - анімації та particle-ефекти вибухів.
4. Додавання аудіо:
   - постріли, влучання, вибухи, музичний ритм через `Web Audio API`.
5. Створення розширеної версії з покращенням дизайну і геймплею:
   - новий дизайн корабля;
   - різні типи ворогів;
   - літаюча тарілка як додатковий противник;
   - покращені візуали куль;
   - система рівнів складності та щити.

## Важливо
- `space-invaders.html` — файл, що стосується базового завдання (класична реалізація гри).
- `space-invaders-nova.html` — розширена версія, у яку додано покращений дизайн, різні типи ворогів, покращені кулі, рівні складності (`Easy/Hard/Very Hard`) і щити (для складних режимів).
